# Brainstorming Prerequisite Skill / 头脑风暴前置技能

![分类](https://img.shields.io/badge/分类-通用-green)

## 项目简介

在任何创造性工作之前必须使用的前置技能——创建功能、构建组件、添加行为或修改逻辑之前，先探索用户意图、需求和设计。防止跳步直接实现。

## 功能特性

- Brainstorming Ideas Into Designs
- Anti-Pattern: "This Is Too Simple To Need A Design"
- Checklist
- Process Flow
- The Process
- After the Design
- Key Principles
- Visual Companion
