# Brainstorming / 创意头脑风暴

![分类](https://img.shields.io/badge/分类-创意-green)

## 项目简介

创造性工作前的防跳步设计技能。当用户需要头脑风暴、创意发散、方案设计时激活。

核心原则：先设计流程再动手，避免跳过关键步骤。

## 工作方法

1. 明确问题和约束条件
2. 列出所有可能方向（不评判）
3. 对每个方向评估可行性
4. 收窄到2-3个最优方案
5. 细化执行步骤

## 适用场景

- 写作前的选题构思
- 产品功能设计
- 营销方案策划
- 技术架构选型
